# MM Racing — Builds
Motorcycle Build Logs & Engineering Notes
This repository contains the real build data, notes, and documentation behind MM Racing projects.
These are not polished articles or marketing posts —
they are working logs from the garage.
Every build is documented as it happens:
What was changed
Why it was changed
What worked
What didn’t
### 🏍️ What This Repo Is
A collection of motorcycle builds, upgrades, and experiments, including:
Track bike builds
Restoration projects
Performance modifications
Custom fabrication
Electronics and telemetry integration (APEXSENSE™)
Each build is broken down into clear, structured notes that can be followed, reused, or adapted.
### 🧠 Philosophy
Most build content online skips the reality.
This doesn’t.
This repo shows the actual process — including mistakes, revisions, and problem-solving.
Because that’s where the real value is.
### 📂 Structure
Each bike/build typically includes:
Overview — what the build is and the goal
Spec Sheet — current configuration
Mod List — completed + planned upgrades
Build Log — chronological progress
Technical Notes — deeper engineering detail
Parts & Sources — what was used and where it came from
### 🔧 Example Projects
Yamaha YZF750R (1994) — Track Build / Development Platform
Yamaha R1 (2003) — Performance Street / Track Hybrid
CRF250R (2004) — Restoration (Moot Mechanic Project 👊)
